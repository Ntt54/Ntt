ÉTAPE 1 — COPIER LE PROJET
────────────────────────────────────────────────────────────
Placer le dossier labo-iut-fotso/ dans :
  C:\xampp\htdocs\labo-iut-fotso\

ÉTAPE 2 — DÉMARRER XAMPP
────────────────────────────────────────────────────────────
Ouvrir XAMPP Control Panel
Démarrer Apache  ✅
Démarrer MySQL   ✅

ÉTAPE 3 — CRÉER LA BASE DE DONNÉES
────────────────────────────────────────────────────────────
1. Ouvrir http://localhost/phpmyadmin
2. Cliquer sur "Nouvelle base de données"
3. Nom : labo_iutfv
4. Interclassement : utf8mb4_general_ci
5. Cliquer sur "Créer"
6. Onglet "Importer"
7. Choisir le fichier database/labo_iutfv.sql
8. Cliquer sur "Exécuter"

ÉTAPE 4 — ACCÉDER AU SITE
────────────────────────────────────────────────────────────
Site public :  http://localhost/labo-iut-fotso/
Panel admin :  http://localhost/labo-iut-fotso/admin/login.php

Identifiants admin par défaut :
  Login    : admin
  Mot de passe : Admin@2025

ÉTAPE 5 — VÉRIFIER LA CONNEXION BDD (Optionnel)
────────────────────────────────────────────────────────────
Ouvrir includes/db.php
Vérifier :
  DB_USER='root'
  DB_PASS='' (par défaut sur XAMPP)

Modifier si votre XAMPP a un mot de passe MySQL différent.
